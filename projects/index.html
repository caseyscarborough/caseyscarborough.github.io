Projects

A collection of projects that I've worked on.


The following is a list of some of the projects I've worked on. The bulk of my work and my portfolio is held at GitHub. In general, I am currently developing with Ruby, Rails, and the Java/Groovy-based Grails web framework. I am always building on my portfolio and this will constantly be updated. For the most up to date info, visit my GitHub page.


  • GitHub API Wrapper
    A simple and easy-to-use wrapper for GitHub's API v3.
  • Yield Gem
    A command-line utility for previewing markdown files in GitHub Flavored Markdown.
  • Slinky Web Application
    A Rails web-application for shortening and keeping track of important URLs.
  • GitHub Activity Stream
    An embeddable widget written in Rails for displaying GitHub activity for a user.
  • libcrypt
    A simple hash and encryption library in C.
  • Java Sliding Puzzle Solver
    A solver for the 8-puzzle game. It demonstrates the A* Search algorithm and using heuristics to find a goal state.
  • GitLab Automated Installation
    A collection of shell scripts for automated installation of GitLab.
  • Instagram API Wrapper
    A wrapper for Instagram's JSON-based API.
  • HomeLink
    A Grails web application used to create to-do lists, shopping lists, and reminders.
  • OMDBAPI Gem
    A very easy to use API wrapper for getting movie and television show information from OMDB's API.
  • Rails Blog
    A full featured blog engine written in Rails.
  • Asteroids
    An implementation of the classic arcade game Asteroids using Java.
